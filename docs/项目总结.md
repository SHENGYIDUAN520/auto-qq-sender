# 自动查寝消息发送系统 - 项目总结

## 项目概述

本项目旨在利用 Mirai 框架实现一个自动发送 QQ 查寝消息的系统，主要包括：
1. Mirai 框架安装与配置
2. HTTP API 接口设置
3. Python 脚本发送消息
4. Windows 任务计划自动执行

## 替代方案

考虑到当前网络环境的限制，我们提供了以下替代方案：

1. **测试模式**
   - 我们添加了测试模式，可以模拟消息发送过程，方便熟悉系统功能
   - 可以通过 `scripts/test_send.ps1` 脚本运行测试

2. **其他框架选择**
   - 如果 Mirai 框架持续无法正常安装，可以考虑:
     - OneBot 框架
     - NoneBot2 框架
     - 使用微信或企业微信机器人替代
